# Escopo

## Escopo contemplado
	- Mapeamento do processo de plantio agrícola do recebimento das sementes ao pós-plantio
	- Identificação de pontos críticos e oportunidades de controle
	- Levantamento e documentação de requisitos
	- Definição de regras de negócio para avaliação de qualidade
	- Proposta de solução sistêmica
	- Definição de indicadores de desempenho e qualidade

## Fora do escopo
	- Desenvolvimento completo da aplicação
	- Integrações com sistemas reais de terceiros
	- Uso de dados reais de empresas
